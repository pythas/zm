# Zero Matter

**Zero Matter** is a pixel-art space survival and engineering game built with Zig and WebGPU. 
The player starts with a precarious, damaged vessel and must physically engage with the environment to understand and replicate technology.

## Development

### Prerequisites
*   [Zig](https://ziglang.org/) (Version 0.14.0 or later)
*   [WebGPU](https://github.com/hexops/mach-gpu) compatible GPU/Drivers

### Building & Running
```bash
zig build run
```
